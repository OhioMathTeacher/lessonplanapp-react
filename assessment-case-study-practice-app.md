# Assessment Case Study Practice App (MVP)

## Purpose
A web-based tool to help students practice analyzing student work for the Assessment Case Study assignment, using both sample artifacts and their own data, with expert “look-fors” and just-in-time support.

## Core Features
- Practice Mode: View 3 sample student work artifacts, enter analysis (strengths, misconceptions, patterns)
- Reveal expert “look-fors” and analysis for comparison
- Upload Mode: Upload own student’s work samples, guided prompts for analysis
- Prompts for: strengths, misconceptions, change over time, class comparison, data limits, next steps
- “ToddGPT” chat window for hints, questions, and feedback

## User Flow
1. Practice Mode: Analyze sample artifacts, compare to expert analysis
2. Upload Mode: Analyze own student’s work, guided by prompts
3. (Optional) Use ToddGPT for support

## Stretch Features
- Export draft memo/outline
- Peer review or instructor feedback
- Rubric checklist integration
