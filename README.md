# Lesson Plan Revision App

## Purpose
A web-based tool to help students organize, upload, and revise their three field lesson plans, with targeted revision ideas and curated resource links for Technology, Differentiation, and Discourse.

## Core Features
- Upload artifacts for each of three lessons (PDF, DOCX, images)
- 3x3 grid of revision ideas (Technology, Differentiation, Discourse for each lesson)
- Clickable boxes for high-level and detailed suggestions
- Curated resource links for each revision area (e.g., YouCubed, MARS, NCTM, course texts)
- Optional: “ToddGPT” chat window for revision Q&A

## User Flow
1. Upload lesson artifacts for each lesson
2. Review and select revision ideas in each focus area
3. Explore resource links for inspiration/examples
4. (Optional) Chat with ToddGPT for feedback

## Stretch Features
- Save/export editable lesson plan
- Peer feedback integration
- Analytics on revision focus areas

## Getting Started

In the project directory, you can run:

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.
